
Créer un site web moderne et professionnel pour le **Lycée Professionnel de Yamoussoukro** en utilisant React, TypeScript, Tailwind CSS, et shadcn/ui.

## 🎨 DESIGN SYSTEM

**Palette de couleurs :**
- Primary: #1a5f2a (Vert forêt - éducation)
- Secondary: #ff6b35 (Orange énergie)
- Accent: #f4a261 (Orange clair)
- Light: #e9c46a (Jaune doré)
- Dark: #264653 (Bleu foncé)
- Background: #ffffff / #f4f4f4
- Text: #2d3436

**Typography:**
- Headings: Inter, bold
- Body: Inter, regular
- Utiliser des tailles responsives

**Style général:**
- Design épuré, moderne, institutionnel
- Animations fluides avec Framer Motion
- Composants glassmorphism pour les cartes
- Icons avec Lucide React
- Responsive mobile-first

## ⚙️ FONCTIONNALITÉS TECHNIQUES

- Routing: Next.js App Router
- Animations: Framer Motion (fade in, slide up, stagger)
- Responsive: Tailwind breakpoints (sm, md, lg, xl)
- Images: Next/Image optimization
- SEO: Metadata par page
- Performance: Lazy loading, code splitting
- Accessibility: ARIA labels, contrast ratios
